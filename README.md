# Estrutura
A ideia é um hub de jogos usando diversas tecnologias, com objetivo de apromoramento de skills, estas sao as skills trabalhadas:

- Acumulação de pontos com memoria flash (Redis).
- Trabalhar com node.js javascript para o backend.
- Utilização de ambiente virtualizado (Docker).

## Backend
- Node.js 
- Express.js
- Redis

## Frontend
- Nuxt.js
- Tailwind.js
